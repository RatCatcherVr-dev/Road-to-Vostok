A dedicated preservation and reverse-engineering project for **Road to Vostok**, the hardcore single-player survival game set in a post-apocalyptic border zone. This repository focuses on analyzing, rebuilding, and decompiling core game frameworks to enable educational study, asset preservation, and custom modding workflows.
